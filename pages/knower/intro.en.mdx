# Introduction

Knower is fashion E-commerce platform that sells vintage clothes which are modern, simple but stylish.
There has been an increasing number of needs of domestic customers for vintage clothes, but they were only few platforms dealing with those products,
and vintage clothes available in South Korea were almost expensive or hard to get access to.

This is the reason why Knower service has been started.

## Tech Stacks

There are two apps for consisting of this project, and each app has different tech stacks used.

1. Admin Page
2. Client Page

### Admin Page

Admin page is a admin panel that only Knower admin users can access.
It is mainly used by Knower admin users for managing various types of operations such as maintaining products, organizing customers' orders and so on.
