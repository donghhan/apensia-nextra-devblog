# 백준 2750번

## 문제

N개의 수가 주어졌을 때, 이를 오름차순으로 정렬하는 프로그램을 작성하시오.

## 입력

첫째 줄에 수의 개수 N(1 ≤ N ≤ 1,000)이 주어진다. 둘째 줄부터 N개의 줄에는 수 주어진다. 이 수는 절댓값이 1,000보다 작거나 같은 정수이다. 수는 중복되지 않는다.

```bash
# 예제 입력
5
5
2
3
4
1
```

## 출력

첫째 줄부터 N개의 줄에 오름차순으로 정렬한 결과를 한 줄에 하나씩 출력한다.

```bash
# 예제 출력
1
2
3
4
5
```

## 풀이

굉장히 단순한 정렬 문제이다. N의 개수 또한 1000개로 그렇게 크지 않으니 문제에서 직접 주어진대로 시간복잡도가 $O(n^2)$인 알고리즘을 사용해볼 수 있다.

<br/>

### 자바스크립트

<a href="https://developer.mozilla.org/en-US/docs/Web/JavaScript/Reference/Global_Objects/Array/sort" target="_blank"><code>sort()</code></a> 메소드를 이용한다. 
다만, 주의할 점은 자바스크립트에선 정렬을 아스키코드 순서대로 정렬을 한다는 점이다.

`sort()`는 `compareFn(a, b)` 비교 함수를 인자로써 받는데 `compareFn()`의 첫번째 인자(`a`)와 두번째 인자(`b`)의 차를 비교하여 만일 음수일 경우 `a`가 `b`보다 앞에 정렬되고, 반대도 마찬가지이다.
만일 같다면 정렬은 이루어지지 않는다.

```js
// 입력값 받기
const input = require("fs")
  .readFileSync(0)
  .toString()
  .trim()
  .split("\n");
// 첫째줄 N과 그 다음줄에 계속 나오는 수들을 nums라는 이름의 배열에 담기
let [N, ...nums] = input.map(Number);

// sort() 메소드
nums = nums.sort((a, b) => a - b);

// 정렬된 배열의 요소들을 반복문을 통해 출력
for (let i of nums) {
  console.log(i);
}
```

<br/>

### 파이썬

List comprehension을 통해 입력값 `N`만큼 정수형으로 입력값을 한 줄에 하나씩 받는다. 이후, <a href="https://docs.python.org/3/howto/sorting.html#sorting-basics" target="_blank"><code>sorted()</code></a> 함수를 통해 오름차순으로 정렬된 새로운 배열을 만들어낸다.

```py
# 입력값 받기
# label 5
N = int(input())
# label [1, 2, 3, 4, 5]
nums = sorted(list(int(input()) for _ in range(N)))

for i in nums:
    print(i)
```
