# CSS 단위들

## `px`

제일 간단한

## 상댓값

### `rem`

<u>**_루트_**</u> 요소에 비례한 크기를 반환한다. 여기서 *루트*란 HTML 문서에서 최상단 부분에 해당하는 [`<html>`](https://developer.mozilla.org/ko/docs/Web/HTML/Element/html) 태그를 의미한다.
기본값으로는 16px로 고정이 되어 있으며 이는 얼마든지 CSS를 통해 바꿀 수 있다.

### `em`

<u>**_부모_**</u> 요소에 비례한 크기를 반환한다.

#### `rem`과 `em`을 사용하는 이유

`px`의 경우 유저가 선호하는 브라우저의 환경 설정은 무시한 채 오직 주어진 고정값만을 집어넣기 때문이다. 밑의 예제를 한 번 보자.

첫번째 `<ul>`은 `font-size`를 `16px`로 고정을 시켜놨고 두번째 `<ul>`은 `font-size`를 `1rem`을 집어넣어 루트 요소에 비례한 상대적인 값을 넣었다.
크롬 브라우저를 예로 들었을 때 설정 &rarr; 모양 &rarr; 폰트 커스터마이징 &rarr; 최상단에 있는 폰트 크기를 통해 기본 폰트 크기를 조절할 수 있다.
기본값은 16px이나 이를 자유자재로 움직였을 때 `px` 값을 지정한 `<ul>` 내부의 `<li>`는 폰트 크기에 변함이 없지만 `rem`을 지정한 `<ul>` 내부의 `<li>`는 폰트 크기에 변함이 있는 걸 알 수 있다.
