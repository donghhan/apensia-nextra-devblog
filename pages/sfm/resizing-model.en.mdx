# Customizing Model Size

## Resizing Model

### Adding Scale Control

First, import your model and mouse right click and select Add Scale Control To Models by accessing Utilities &rarr; Add Scale Control To Models.

![importing_model](../../media/sfm/scaling-model-size/importing-model.png)

### Resizing rootTransform_scale

You can now resize your model with newly added rootTransform_scale key.

![rootTransform_scale](../../media/sfm/scaling-model-size/roottransform_scale.png)

<br />

## Resizing Individual Bone

This feature is newly added into S2FM.

### Entering Model Element Viewer

Mouse right click on your imported model and click Show in Element Viewer &rarr; Model in order to get into Element Viewer.

<br />

<center>
  ![model_element_viewer](../../media/sfm/scaling-model-size/model_element_viewer.png)
</center>

<br />

### Including Float Variable For Bone

Expand bones menu and locate to bone that you want to resize. Right click on the bone and choose Add Attribute &rarr; float.

<br />

<center>
  ![bone_float](../../media/sfm/scaling-model-size/bone_float.png)
</center>

<br />

### Adding Scale Variable

Create new float type variable called `scale`.

<br />

<center>
  ![adding_scale](../../media/sfm/scaling-model-size/adding_scale.png)
</center>

<br />

### Resizing Bone

Then you may notice that your character's head suddenly disappeared.

<br />

<center>
  ![scale_zero](../../media/sfm/scaling-model-size/scale_zero.png)
</center>

<br />

Don't worry. This is because the initial value is set to be 0, so you can simply change value of `scale`.
Your model will get as bigger as the value that you input (e.g. input 2 will makes your model twice as bigger, 10 as 10 times bigger).

<br/ >

<center>
  ![scale_three](../../media/sfm/scaling-model-size/scale_three.png)
</center>
