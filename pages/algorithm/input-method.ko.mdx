# 다양한 입력값 받기

코딩 테스트를 준비하면서 몇몇 사이트들은 입력값 처리를 따로 해줄 필요가 없었지만
다른 몇몇 사이트들은 입력값까지 직접 받아야 하는 경우가 생겼다.
그래서 이번 기회에 다양한 입력값을 어떻게 받을 수 있는지 나름 정리를 해보았다.
이 포스트에선 파이썬과 자바스크립트 방식을 나란히 정리해보았다.

<br/>

## 한 줄에 입력값 받기

한 줄에 입력값을 받는 것은 각 변수별로 지정해주면 된다.

### 기본

파이썬에서의 입력값 받기는 굉장히 쉽다. 단순히 `input()`을 사용하면 된다. 다만, 그냥 `input()`을 사용하게 될 경우 어떤 값이 들어가게 되든 문자열로 받게 되니
필요에 따라 형변환을 강제로 해주면 된다.

자바스크립트는 `prompt()`를 사용하면 된다. 자바스크립트도 파이썬과 마찬가지로 단순히 `prompt()`로만 입력값을 받게 되면 문자열로 받게 되니
필요에 따라 형변환을 해주면 된다.

👉 **_각각의 출력문 `print()` 및 `console.log();` 부분에 마우스 오버를 하여 출력값을 볼 수 있다._**

<CH.Code>

```py main.py
text = input() # 형변환을 안 해줄 경우 무조건 문자열로 받게 된다.
# label 1 <class 'str'>
print(text, type(text))

a = int(input()) # 정수형으로 변환
# label 1 <class 'int'>
print(a, type(a))

b = str(input()) # 문자열로 변환
# label abc <class 'str'>
print(b, type(b))

c = float(input()) # 실수형으로 변환
# label 1.51 <class 'float'>
print(c, type(c))
```

---

```js main.js
const text = prompt(); // 형변환을 안 해줄 경우 무조건 문자열로 받게 된다.
// label 1 string
console.log(text, typeof text);

const a = Number(prompt()); // 정수 및 실수형으로 변환
// label 1 number
console.log(a, typeof a);

const b = String(prompt()); // 문자열로 변환
// label abc string
console.log(b, typeof b);
```

</CH.Code>

<br/>

### 더 효율적인 방법

상기 방식에서 여러 입력값을 받게 될 경우 시간 초과가 발생할 수 있기 때문에 다른 방식으로 해볼 필요가 있다.

파이썬에서는 <a href="https://docs.python.org/3/library/sys.html#sys.stdin" target="_blank"><code>sys.stdin</code></a>을 사용하여 입력값을 받고
각 줄의 데이터를 읽어들이는 <a href="https://docs.python.org/3/library/readline.html" target="_blank"><code>readline</code></a> 모듈을 이용한다.

자바스크립트에서는 크게 다음과 같이 두 가지 방식이 존재한다.

1. <a href="https://nodejs.org/dist/latest-v16.x/docs/api/readline.html#readline" target="_blank"><code>readline</code> 모듈</a>을 이용하는 방법
2. <a href="https://nodejs.org/dist/latest-v16.x/docs/api/fs.html#file-system" target="_blank"><code>fs</code> 모듈</a>을 이용하는 방법

보통의 경우 `fs` 모듈을 이용하는 것이 `readline` 모듈을 이용할 때보다 더 빠르기 때문에 이번 포스트에서는 `fs` 모듈을 이용하는 방식으로 진행해보겠다.
`fs` 모듈 내에는 <a href="https://nodejs.org/dist/latest-v16.x/docs/api/fs.html#fsreadfilesyncpath-options" target="_blank"><code>readFileSync()</code></a> 메소드가 있는데
인자(문자열) 경로에 있는 파일의 콘텐츠를 읽어들인 후 객체 타입으로 반환을 해준다 (예제에서는 `data.txt`라는 텍스트 파일 안에 데이터를 집어넣었다는 가정 하에 진행한다). 이를 <a href="https://developer.mozilla.org/en-US/docs/Web/JavaScript/Reference/Global_Objects/Object/toString" target="_blank"><code>toString()</code></a> 메소드를 사용하여
객체로 반환된 것을 문자열로 치환해주고 마지막으로 <a href="https://developer.mozilla.org/en-US/docs/Web/JavaScript/Reference/Global_Objects/String/Trim" target="_blank"><code>trim()</code></a> 메소드를 이용하여
문자열의 양 끝 공백을 제거해준다.

해당 부분은 문자열로 치환이 되었으니 숫자로 변경을 하고 싶을 경우 <a href="https://developer.mozilla.org/en-US/docs/Web/JavaScript/Reference/Global_Objects/Number" target="_blank"><code>Number()</code></a>로 감싸주면 된다.

👉 **_각각의 출력문 `print()` 및 `console.log();` 부분에 마우스 오버를 하여 출력값을 볼 수 있다._**

<CH.Code>

```py main.py
import sys

input_int = int(sys.stdin.readline()) # 정수형으로 형 변환
# label 1 <class 'int'>
print(input_int, type(input_int))

input_str = str(sys.stdin.readline()) # 문자열로 형 변환
# label abc <class 'str'>
print(input_str, type(input_str))
```

---

```js main.js
// 형 변환을 하지 않을 경우 문자열로 반환된다.
const input = require('fs').readFileSync('data.txt').toString().trim();
// label 1 string
console.log(input, typeof input);

// 숫자로 형 변환을 할 경우
const inputInNumber = Number(require('fs').readFileSync('data.txt').toString().trim());
// label 1 number
console.log(inputInNumber, typeof inputInNumber);
```

</CH.Code>

<Callout>
    파이썬에서 형변환을 거치지 않고 단순히 `a = sys.stdin.readline()`을 하게 되면 개행 처리 문자(`\n`)까지 받게 되므로, 반드시 형변환을 거친 후 사용하도록 하자. 
</Callout>

## 한 줄에 공백 기준으로 여러 입력값 받기

입력되는 값들은 아래와 같다.

```bash
# 입력
1 2 3 4 5
```

<br/>

<span class="logo"><SiPython/></span>

<a href="https://docs.python.org/3/library/stdtypes.html?highlight=split#str.split" target="_blank"><code>split()</code></a>을 이용하여 공백 기준으로 문자열을 구분한다.
이후, <a href="https://docs.python.org/3/library/functions.html#map" target="_blank"><code>map()</code></a> 내장 함수를 활용한다.
`map()`은 반복되는 값(`interable`)에 대하여 어떤 함수(`function`)를 적용하여 새로운 `map` 객체를 반환해주는 역할을 한다.
이후에 <a href="https://docs.python.org/3/library/functions.html#func-list" target="_blank"><code>list()</code></a>를 이용해 해당 `map` 객체를 배열 안에 담아준다.

<br/>
<br/>

<span class="logo"><SiJavascript/></span>

<a href="https://developer.mozilla.org/en-US/docs/Web/JavaScript/Reference/Global_Objects/String/split" target="_blank"><code>split()</code></a>를 이용하여 구분을 할 수 있다.
이를 숫자형으로 형변환하기 위해서는 파이썬의 `map()` 메소드와 같이 자바스크립트에서도 존재하는 <a href="https://developer.mozilla.org/en-US/docs/Web/JavaScript/Reference/Global_Objects/Array/map" target="_blank"><code>map()</code></a> 메소드를 이용한다.
`map()`의 첫번째 인자로 적용을 시킬 콜백 함수를 받게 된다. 각각의 `el`에 대하여 <a href="https://developer.mozilla.org/en-US/docs/Web/JavaScript/Reference/Global_Objects/parseInt" target="_blank"><code>parseInt()</code></a> 메소드를 이용해 각 원소들을 숫자형으로 치환해준다.

<CH.Code>

```py python
import sys

data_int = list(map(int, sys.stdin.readline().split())) # 정수형으로 형변환
# label [1, 2, 3, 4, 5] <class 'list'>
print(data_int, type(data_int))

data_str = list(map(str, sys.stdin.readline().split())) # 문자형으로 형변환
# label ['a', 'b', 'c', 'd', 'e'] <class 'list'>
print(data_str, type(data_str))
```

---

```js javascript
// 기본: 문자열
const inputInString = require("fs")
  .readFileSync("data.txt")
  .toString()
  .trim()
  .split(" ");
// label ['1', '2', '3', '4', '5'] object
console.log(inputInString, typeof inputInString);

// 숫자로 형변환
const inputInNumber = require("fs")
  .readFileSync("data.txt")
  .toString()
  .trim()
  .split(" ")
  .map((el) => parseInt(el));
// label [1, 2, 3, 4, 5] object
console.log(inputInNumber, typeof inputInNumber);
```

</CH.Code>

> ### 🙋‍♂️ `Number()`와 `parseInt()`의 차이점 [^1]
>
> `Number()`의 경우 타입을 변경하지만 `parseInt()`의 경우 입력값을 파싱하여 숫자로 변환시켜준다. 예를 들어,
>
> ```js
> // 타입 변경
> Number('123hello'); // NaN
> Number('5e1'); // 50 👉 Numeric Literals 참조
> ```
>
> `Number()`의 경우 인자 안의 문자열의 타입을 숫자형으로 바꿔주되, 만일 인자 안의 문자열 중 숫자로 변환될 수 있는 것들이 없는 경우 `NaN`을 반환한다.
> `5e1`의 경우 numeric literal의 제곱 법칙을 따라 문자열이지만 숫자형으로 인식하여 해당 값을 숫자형으로 반환시켜준다 <a href="https://developer.mozilla.org/en-US/docs/Web/JavaScript/Reference/Lexical_grammar#numeric_literals" target="_blank">&rarr; Numeric Literal의 exponential 부분 참조</a>
>
> <br/>
>
> `Number()` 대신 연산자 `+`를 앞에 써도 똑같은 결과값을 가져오게 된다.
>
> ```js
> console.log(+'123hello'); // NaN
> console.log(+'5e1'); // 50
> ```
>
> <br/>
>
> 반면 `parseInt()`의 경우, 인자 안의 문자열 중 숫자로 되어 있는 것들만 찾아서 숫자값으로 반환해준다.
>
> ```js
> // 파싱하여 숫자로 변경
> parseInt('32px'); // 32
> parseInt('5e1'); // 5
> ```

[^1]: 참조: <a href="https://thisthat.dev/number-constructor-vs-parse-int/" target="_blank"><code>Number()</code> vs <code>parseInt()</code></a>

## 여러 줄에 걸쳐 입력값 받기

이번에는 여러 줄에 입력값을 받는 법에 대해 알아보겠다. 여러 줄에 걸쳐 입력값을 받는 방법에는 크게 두가지 방법이 있을 것이다.

1. 무한정 입력값을 받는 방법
2. 테스트 케이스(T)만큼 입력값을 받는 방법

<br/>

### 입력값을 무한정 받기

필요에 따라선 입력값을 정말 무한정 받을 순 있겠지만 대체적으로 코딩 테스트에서는 특정 입력값을 넣게 되면 종료가 되는 부분이 있다.

<br/>

### 테스트 케이스 줄만큼 입력값 받기

대부분의 입력값을 받는 행위는 아마도 이렇게 받을 것이다. 입력값은 다음과 같다.

```bash
# 테스트 케이스 (자연수)
5
# 이후 각 줄에 테스트 케이스 수만큼 하나씩 입력값 받음
1
2
3
4
5
```

<br/>

<span class="logo"><SiPython/></span>

테스트 케이스는 일반적으로 자연수로 주어지기 때문에 간단하게 `int(input())`으로 받아도 무난하다.

아래 `input_str`의 경우 다음 순서대로 코드를 작성하였다.

1. `for _ in range(T)`: 테스트 케이스(`T`)만큼만 반복하면 되고 반복되는 값이 필요 없으므로 `_`로 받아주었다. 그리고 <a href="https://docs.python.org/3/library/stdtypes.html?highlight=range#range" target="_blank"><code>range()</code></a>를 이용하여 인덱스 범위를 지정해주었다.
2. 입력값은 위에서 살펴본 대로 `str(sys.stdin.readline().rstrip())` 코드를 가지고 `T`만큼 한 줄에 하나씩 입력값을 받는다. <a href="https://docs.python.org/3/library/stdtypes.html?highlight=rstrip#bytearray.rstrip" target="_blank"><code>rstrip()</code></a>을 사용한 이유는 `readline()`을 쓰게 되면 오른쪽에 자동으로 딸려오는 개행 문자 `\n`을 없애주기 위함이다.
3. 이를 전체 `[]`로 감싸서 배열 안에 담아준다. 이를 파이썬에서는 <a href="https://docs.python.org/3/tutorial/datastructures.html?highlight=list%20comprehension#list-comprehensions" target="_blank">List Comprehension</a>이라 부른다.

👉 **_각각의 출력문 `print()` 부분에 마우스 오버를 하여 출력값을 볼 수 있다._**

```py
import sys

T = int(input())

# 문자열로 입력값 받기
input_str = [str(sys.stdin.readline().rstrip()) for _ in range(T)]
# label 5 ['1', '2', '3', '4', '5']
print(T, input_str)
# label <class 'list'> <class 'str'>
print(type(input_str), type(input_str[0]))

# 정수형으로 입력값 받기
input_int = [int(sys.stdin.readline().rstrip()) for _ in range(T)]
# label 5 [1, 2, 3, 4, 5]
print(T, input_int)
# label <class 'list'> <class 'int'>
print(type(input_int), type(input_int[0]))
```

<br/>

<span class="logo"><SiJavascript/></span>

1. 자바스크립트에서는 위에서 봤던 것과 거의 유사하지만 맨 마지막의 `split()`의 인자로 개행 문자열 `\n`을 넣어줌으로서 각 입력값이 한 줄에 입력될 수 있도록 한다.
2. 이후 상수 `T`와 `arr`을 선언하여 `input`에 있는 값들을 <a href="https://developer.mozilla.org/en-US/docs/Web/JavaScript/Reference/Operators/Destructuring_assignment" target="_blank">구조 분해 할당</a>을 통해 각각의 값들을 담는다.
3. 이때, `arr` 안의 값들을 <a href="https://developer.mozilla.org/en-US/docs/Web/JavaScript/Reference/Operators/Spread_syntax" target="_blank">Spread Operator</a>를 이용하여 내부 값들을 꺼내온다.

👉 **_각각의 출력문 `console.log();` 부분에 마우스 오버를 하여 출력값을 볼 수 있다._**

```js
// 문자열로 변환
const input = require('fs').readFileSync('data.txt').toString().trim().split("\n");
const [T, ...arr] = input;
// label 5 [ '1', '2', '3', '4', '5' ]
console.log(T, arr);
// label object
console.log(typeof arr);

// 숫자형으로 변환
const inputInNumber = require("fs")
  .readFileSync("data.txt")
  .toString()
  .trim()
  .split("\n")
  .map((el) => parseInt(el));
const [T2, ...arrTwo] = inputInNumber;
// label 5 [ 1, 2, 3, 4, 5 ]
console.log(T2, arrTwo);
// label object
console.log(typeof arrTwo);
```

#### 첫째줄엔 테스트 케이스 두번째 줄에 공백으로 여러 개

입력의 예시는 다음과 같다.[^2]

[^2]:
    <a href="https://www.acmicpc.net/problem/10818" target="_blank">백준 10818번</a>

```bash
5 # 테스트 케이스 수
20 10 35 30 7 # 테스크 케이스 수만큼 받은 데이터 양
```

<br/>

<span class="logo"><SiJavascript/></span>

1. 맨 처음에 파일을 불러온 후 `.split()` 메소드로 개행 문자 별로 나눈 다음,
2. 그 중 첫번째 요소는 `Number()`로 감싸주고,
3. 두번째 요소는 다시 `.split()`으로 공백을 기준으로 데이터를 나눈 후, `.map()`으로 각 요소들을 숫자형으로 치환한다.

👉 **_각각의 출력문 `console.log();` 부분에 마우스 오버를 하여 출력값을 볼 수 있다._**

```js main.js
const input = require("fs")
  .readFileSync("data.txt")
  .toString()
  .trim()
  .split("\n");
const N = Number(input[0]);
let nums = input[1].split(" ").map(Number);

// label 5 [ 20, 10, 35, 30, 7 ]
console.log(N, nums);
// label number object
console.log(typeof N, typeof nums);
```

### 주의 (예외 케이스)

가끔씩 예외를 하게 될 수도 있다.
