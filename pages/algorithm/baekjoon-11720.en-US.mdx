# Baekjoon 11720

## 문제

Given N numbers without whitepaces, print the sum of all numbers.

<br/>

## Input and Output

N (1 ≤ N ≤ 100) is given at the first row, followed by numbers without whitespaces.

<CH.Code>

```bash input1
1
1
```

```bash output1
1
```

</CH.Code>

<CH.Code>

```bash input2
5
54321
```

```bash output2
15
```

</CH.Code>

<CH.Code>

```bash input3
25
7000000000000000000000000
```

```bash output3
7
```

</CH.Code>

<CH.Code>

```bash input4
11
10987654321
```

```bash output4
46
```

</CH.Code>

<br/>

## Solution

As there is no whitespace, we need to divide them first.

1. Divide input numbers one by one and put them in a new array.
2. Convert types of all elements in the array from String to Number.
3. Print the sum of all elments.

<br/>

### JavaScript

First, we will use <a href="https://developer.mozilla.org/en-US/docs/Web/JavaScript/Reference/Global_Objects/String/split" target="_blank"><code>split(separator)</code></a> built-in method for String.
`separator` will be a separator for dividing strings. In this case, as there are **_no whitespaces_**, we can pass `""`.

```js
const input = require("fs")
  .readFileSync(0)
  .toString()
  .trim()
  .split("\n");

const N = Number(input[0]); // test case N
const nums = input[1].split(""); // Put all numbers in an array called nums

let answer = 0; // Declare variable to save sum value

for (let i = 0; i < N; i++) { // Through iterations,
  answer += Number(nums[i]); // convert all elements' types from String to Number and add them up to answer
}

console.log(answer);
```

<br/>

### Python

Using built-in method of `map()`, convert all types of inputs to Number and print sum value with using built-in method of `sum()`.

```py
import sys

N = int(sys.stdin.readline())
print(sum(map(int, sys.stdin.readline().rstrip())))
```
