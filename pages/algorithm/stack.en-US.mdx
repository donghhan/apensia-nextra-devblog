## Definition

Stack is linear data structure with Last In, First Out(LIFO), which means element that has been inserted lastly will come out first.
The best example for this is a box.

## Time Complexity

1. As data will be added or deleted on the very top, time complexity will always be $O(1)$.
2. For search, as we need to repeat ourselves until we find the element, time complexity will be $O(n)$.

## Practices

### With using array(list)

#### JavaScript

With using <a href="https://developer.mozilla.org/en-US/docs/Web/JavaScript/Reference/Global_Objects/Array/push" target="_blank"><code>push()</code></a> method, we can add element one by one beginning from the back of the empty array called stack.
For deleting element, we can use <a href="https://developer.mozilla.org/en-US/docs/Web/JavaScript/Reference/Global_Objects/Array/pop" target="_blank"><code>pop()</code></a> method to delete element from the back too.

It is quite easy for configuring stack in JavaScript as we don't need any logics for inserting elements at the middle of the array.

```js
const stack = [];

// Push
stack.push(1);
stack.push(2);
stack.push(3);
// label [1, 2, 3]
console.log(stack);

// Pop
stack.pop();
// label [1, 2]
console.log(stack);

// Getting value of top
// label 2
console.log(stack[stack.length - 1]);
```

#### Python

Python also has built-in method as JavaScript.
We can use <a href="https://docs.python.org/3/library/array.html?highlight=append#array.array.append" target="_blank"><code>append()</code></a> method for inserting element from the back
and use <a href="https://docs.python.org/3/library/array.html?highlight=pop#array.array.pop" target="_blank"><code>pop()</code></a> for deleting element from the back, respectively.

```py
stack = []

# Append
stack.append(1);
stack.append(2);
stack.append(3);
# label [1, 2, 3]
print(stack)

# Pop
stack.pop()
# label [1, 2]
print(stack)

# Getting value of top
# label 2
print(stack[-1])
```

### With Using Linked List

#### JavaScript

To begin with, I created `Node` class consisting of the linked list.

`push(value)` method puts `value` in the new node, whereas `pop()` deletes value from the node.
Lastly, `size()` method returns the total size of the stack.

```js
class Node {
    constructor(value) {
        this.value = value;
        this.next = null;
    }
}

class Stack {
    constructor() {
        this.top = null;
        this.size = 0;
    }

    push(value) {
        const node = new Node(value);
        node.next = this.top;
        this.top = node;
        this.size += 1;
    }

    pop() {
        const value = this.top.value;
        this.top = this.top.next;
        this.size -= 1;
        return value;
    }

    size() {
        return this.size;
    }
}

const stack = new Stack();

stack.push(1);
stack.push(2);
stack.push(3);
// label 3
console.log(stack.pop());
stack.push(4);
// label 4
console.log(stack.pop());
// label 2
console.log(stack.pop());
```

#### Python

Logic structure is as same as the JavaScript one.

```py
class Node:
    def __init__(self, item, next):
        self.value = value
        self.next = next

class Stack:
    def __init__(self):
        self.top = None
        self.size = 0

    def push(self, value):
        self.next = Node(value, self.next)

    def pop(self):
        self.value = self.top.value
        self.next = self.top.next
        return value
```
