# 거품 정렬

비교식 정렬 중에서 가장 대표적인 정렬 방법 중 하나이다. 정렬되는 방식이 마치 거품이 올라오는 것처럼 보인다하여 거품 정렬이라는 이름이 붙여졌다.

<br/>

## 원리

서로 인접한 두 요소를 검사하여 정렬한다.
오름차순으로 정리한다고 가정하였을 때 인접한 두 요소 중 왼쪽의 수가 오른쪽의 수보다 작을 경우 그대로 두지만 반대의 경우 자리를 바꾼다.
그렇게 하여 모든 원소에 대하여 반복 작업을 한다.

<br/>

## 시간 복잡도

따라서 시간복잡도는 $O(n^2)$이다.
