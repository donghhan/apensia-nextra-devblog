# 백준 2675번

## 문제

N개의 숫자가 공백 없이 쓰여있다. 이 숫자를 모두 합해서 출력하는 프로그램을 작성하시오.

<br/>

## 입력 및 출력

첫째 줄에 숫자의 개수 N (1 ≤ N ≤ 100)이 주어진다. 둘째 줄에 숫자 N개가 공백없이 주어진다.

<CH.Code>

```bash input1
1
1
```

```bash output1
1
```

</CH.Code>

<CH.Code>

```bash input2
5
54321
```

```bash output2
15
```

</CH.Code>

<CH.Code>

```bash input3
25
7000000000000000000000000
```

```bash output3
7
```

</CH.Code>

<CH.Code>

```bash input4
11
10987654321
```

```bash output4
46
```

</CH.Code>

<br/>

## 풀이

입력값이 공백이 없이 주어지기 때문에 이를 분리를 시켜줘야 할 필요가 있다.

1. 입력값들을 하나씩 분리해서 배열에 담은 후
2. 분리된 문자열들을 전부 정수형(숫자형)으로 치환한 다음,
3. 배열 내 모든 원소들의 합을 구해주면 된다.

<br/>

### 자바스크립트

먼저 문자열 내장 메소드인 <a href="https://developer.mozilla.org/en-US/docs/Web/JavaScript/Reference/Global_Objects/String/split" target="_blank"><code>split(separator)</code></a>을 사용한다.
`separator`는 문자열을 나누는 기준점을 넣는 곳이다. 이 문제에서 입력값들이 전부 공백이 **_없이_** 주어졌기 때문에 `""`를 `separator`로써 넘겨주었다.

```js
const input = require("fs")
  .readFileSync(0)
  .toString()
  .trim()
  .split("\n");

const N = Number(input[0]); // 테스트 케이스 N
const nums = input[1].split(""); // 나머지 문자들을 nums라는 배열에 담음.

let answer = 0; // 합을 담아줄 변수로 answer를 0의 값에 할당

for (let i = 0; i < N; i++) { // 반복문을 통해
  answer += Number(nums[i]); // nums의 모든 원소들을 순회하여 숫자형으로 치환해준 다음 answer 값에 더함
}

console.log(answer);
```

<br/>

### 파이썬

`map()` 메소드를 이용하여 모든 입력값들을 숫자형으로 변환해준 후, 이를 `sum()` 메소드를 이용하여 합을 구해준다.

```py
import sys

N = int(sys.stdin.readline())
print(sum(map(int, sys.stdin.readline().rstrip())))
```
