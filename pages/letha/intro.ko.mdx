# 개요

## 서비스 소개

Letha는 태국의 웰니스 여행 프로그램을 유저들이 자유롭게 검색하고 소비할 수 있는 커뮤니티형 플랫폼 서비스이다.

## 서비스 시작 계기

현재 태국의 웰니스 여행 산업이 가지고 있는 문제점들이 무엇이 있는지 분석해보았다.

1. 태국이란 나라가 아직 생소한 여행객들의 경우 무엇을 어디서부터 시작해야할지 갈피조차 못 잡는 경우가 많다.
2. 여행객들은 항상 여행 위험에 노출되어 있다 (사기, 안전사고 등)
3. 웰니스에 특화된 여행들을 검색하는 데에 시간이 많이 걸릴 뿐더러 서비스의 품질조차 보증되어 있지 않은 경우가 많다.

위와 같은 문제점들을 해결하기 위하여 Letha 서비스를 시작하게 되었다.

1. 보안성: Django 안에 이미 내장된 여러 보안성을 강화하는 툴을 쉽게 가져다 쓸 것이냐 아니면 Node.js로 직접 일일히 구현을 하느냐의 차이일 뿐 적절한 npm 라이브러리와 로직 구현만 올바르게 선택한다면 Node.js로 Django에 뒤쳐지지 않는 훌륭한 프레임워크라 생각했다.
2. 개발 속도: Django가 확실히 개발 속도는 빠를 순 있지만 아직까지 필자에겐 충분한 시간이 있기 때문에 크게 와닿진 않은 요소였다.

따라서 결국 Node.js (Express.js)를 선택했다.
