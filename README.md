This is an example website using Nextra (v3) and Code Hike.

- [Code Hike](https://codehike.org) - learn about Code Hike
- [Nextra](https://nextra.site) - learn about Nextra

Clone it with:

```bash
$ npx degit code-hike/examples/with-nextra my-docs
$ cd my-docs
```
